The QC dashboard provides an at-a-glance view of the reviewer's workload across both QC stages.

#### Statistics Cards
- **Total Queue** — Combined count of all sessions in QC1 and QC2 stages
- **Pending Review** — Sessions awaiting initial review across both queues
- **Issues Found** — Sessions that have been flagged with problems
- **Total Reviewed** — Sessions already advanced past QC stages

#### Queue Previews
Two cards show the top 5 sessions in each queue (QC1 and QC2). Each entry displays the topic title, project name, language pair, speaker count, duration, and a status badge (Pending or Issues). A 'View all' link navigates to the full task page.
