The typical annotator workflow is:
1. Open a pending session from the task list
2. Listen to the mixed audio and reference individual speaker tracks as needed
3. Use 'AI Transcribe' to auto-populate turns from the reference transcript, or add turns manually
4. Edit each turn: correct the speaker, language, timestamps, and transcription text
5. Submit for QA review
6. If QA sends back feedback, review the flagged turns (highlighted in amber), read QA comments, make corrections, and resubmit
