The Workflow Builder is the most sophisticated component of the Polivox platform. It provides a visual, node-and-edge canvas for designing the processing pipeline that each recording session will follow. Built on React Flow (@xyflow/react v12), it supports drag-and-drop node creation, multi-path conditional branching, per-node staff assignment, and comprehensive property configuration.
