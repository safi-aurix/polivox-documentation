The Speaker Management page (/speakers) allows the [[Project Manager (PM)]] to maintain a comprehensive database of [[Speaker (SP)]].

#### Table Display
- Avatar initials, Name, Email, Languages (L1/L2/L3 badges), Country, Gender, Star Rating, Flag button

#### Filtering
- Search by name or email
- Filter by country, language, star rating (1-5), flag status (All / Flagged / Not Flagged)

#### CRUD Operations
- **Add Speaker** — Manual entry (name, email, languages, country, gender, rating) or bulk import from .xlsx/.csv
- **View Speaker** — Detail modal showing all info with in-modal rating adjustment, flag toggle, and edit/delete actions
- **Edit Speaker** — Form modal to modify all speaker fields
- **Delete Speaker** — Confirmation dialog with speaker name shown
- **Flag Speaker** — Toggle flag status for performance tracking
- **Rate Speaker** — Interactive 5-star rating widget (1-5 scale)
