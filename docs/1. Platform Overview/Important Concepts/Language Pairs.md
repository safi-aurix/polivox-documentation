In **Polivox**, a **Language Pair** is the foundational setting for any project, defining the linguistic scope of the audio data to be collected.

**Core Structure**
A language pair consists of two components:
- **Source Language:** The primary language of the conversation.
- **Target Language:** The secondary language into which speakers may code-switch or translate during the session.

**Matching Requirement**
The language pair acts as a "filter" for the [[Speaker Matching Algorithm]]. For a speaker to be eligible for a session, their **Language Profile** must include both languages in the pair.
- _Example:_ If a project is set to **EN-ES** (English-Spanish), only speakers verified in both English and Spanish will appear in the candidate pool.
