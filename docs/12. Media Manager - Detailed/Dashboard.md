#### Statistics Cards
- **Total Queue** — Combined count of all sessions in initial and noise processing stages
- **Pending** — Sessions awaiting initial processing or noise injection
- **Issues / Rework** — Sessions flagged by QC that need reprocessing
- **Completed Today** — Sessions successfully processed and submitted today

#### Queue Previews
Two cards display the initial processing queue and noise injection queue, each showing up to 5 sessions with topic, project, speakers, languages, and status (Pending or Issues). 'View all' links navigate to the respective full task pages.
