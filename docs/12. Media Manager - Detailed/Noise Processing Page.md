This page is used for adding background noise to approved 2-speaker recordings.

#### Page Header & Responsibilities
- Select appropriate environmental noise profiles (street, cafe, office, etc.)
- Mix background noise into the clean audio at realistic levels
- Ensure noise doesn't mask or distort speaker voices
- Preview the final mix before submitting
- Submit noise-processed audio to QC2 for validation

![[noise_processing_sessions.png]]
### **Noise Processing Detail View**
The detail modal provides a noise mixing workspace.

**_QC2 Feedback Alert_**
If this session is returning from QC2 with issues, the QC2 reviewer's comments are displayed in an amber alert.

**_Clean Mixed Audio_**
The approved clean recording is shown with a waveform player for reference, with download and replace options.

**_Background Noise Layer_**
The MM selects a noise profile using one of two methods:
- **Preset Noise Profiles** — Six built-in options displayed as a button grid: Cafe, Street, Office, Public Transport, Restaurant, and Market. Only one can be selected at a time. Presets are disabled when a custom file is uploaded.
- **Custom Noise Upload** — A file upload area accepts any audio file as a custom noise layer. When a custom file is selected, its filename is displayed and the preset buttons become dimmed.

Below the noise source selection, an intensity slider (0–100%) controls how much background noise is mixed into the recording.

**_Live Preview_**
A preview card with a waveform player shows the combined clean audio + noise output. It displays the selected noise type and intensity level, allowing the MM to evaluate the result before submitting.

**_Submit Action_**
'Apply Noise & Send for QC2' submits the noise-processed session for quality review.
