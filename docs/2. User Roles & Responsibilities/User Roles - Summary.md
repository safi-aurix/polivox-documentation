Polivox defines six specialized user roles. Each is responsible for a distinct phase of the speech data production pipeline. The table below gives a quick comparison; the sections that follow describe each role's pages and workflows in full detail.

|**Role**|**Key Responsibility**|**Pages**|
|---|---|---|
|Project Manager|Oversee projects, manage teams, review issues, export|Dashboard, Projects, Speakers, Staff, Save to Drive|
|Quality Checker|Two-stage audio quality review (QC1 + QC2)|Dashboard, Initial QC, Noise QC|
|Speaker|Record bilingual conversations|Dashboard, Sessions, Availability, Onboarding|
|Media Manager|Audio processing and noise injection|Dashboard, Initial Processing, Noise Processing|
|Annotator|Transcribe and annotate approved audio|Dashboard, Annotation Tasks|
|Annotator QA|Review annotation accuracy and completeness|Dashboard, QA Tasks|
