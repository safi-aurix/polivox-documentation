Nodes representing human roles can have staff members assigned to them. This is how project managers allocate team members to specific workflow stages.

#### Eligible Node Types
- [[Media Manager (MM)]]
- [[Quality Checker (QC)]]
- [[Annotator (AN)]]
- [[Annotator QA (AQ)]]

#### Assignment Process
- The property panel shows an 'Assign Role' button for eligible nodes
- Clicking opens a searchable dialog showing staff from the database filtered by matching role
- Each staff member card displays: initials avatar, name, email, star rating, language badges
- Already-assigned staff show an 'Assigned' badge
- Clicking a staff card toggles their assignment on/off
- Multiple staff can be assigned to a single node
- Assigned staff appear in the property panel with individual remove buttons
- The node card preview shows the first 2 assigned names and '+N more' for additional staff
