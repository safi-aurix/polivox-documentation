The Edit Project dialog provides a tabbed interface for modifying project configuration:
- **Details Tab** — Edit project description (name is read-only after creation)
- **Language Pairs Tab** — Add new pairs, edit existing pairs (both languages), remove pairs. Removing a pair also removes it from all topic-LP assignments.
- **Topics Tab** — Add new topics, delete topics, toggle language pair assignments per topic. Language pair badges can be clicked to toggle assignment for each topic individually.
