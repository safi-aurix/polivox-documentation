The project list view displays all projects in a searchable, filterable table. Each row shows the project name, ID, assigned staff count, session progress (completed/total with progress bar), completion percentage, and creation date.

#### Filtering & Search
- Text search by project name or ID
- Filters
- Pagination with configurable page size (default 10 per page)

#### Actions (Row Dropdown Menu)
- Edit Project — opens the Edit Project dialog
- Manage Staff & Speakers — opens the team management dialog
- Export to Drive — visible only for projects with completed sessions
