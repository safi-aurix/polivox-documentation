When a project is selected from the project list, the session list view displays all sessions for that project organized in three tabs:
- **Active Tab** — Sessions currently in progress (not exported, not in review). Shows the normal pipeline flow.
- **PM Review Tab** — Sessions that have been flagged with issues at any QC or annotation stage (qc1-issues, qc2-issues, annotation-issues). These require PM disposition.
- **Completed Tab** — Sessions that have been exported to Google Drive.

### **Session Table Columns**
- Topic title, Category (Natural Conversation / Scenario-Based / Interview Style)
- Languages (Language A — Language B)
- Participants (avatar circles with initials, +N overflow)
- Duration (in seconds), Status badge (color-coded), Recorded date
