    Dashboard

    Statistics Cards

    • Live Topics — Number of topics currently available for recording (with open speaker slots)
    • Upcoming — Scheduled sessions the speaker is expected to join
    • Completed — Total number of successfully recorded sessions
    • Needs Re-record — Sessions flagged by QC that require re-recording (shown in red if > 0)

    Responsibilities Box

    A role information box displays the speaker's core duties:

    • Join the recording room at scheduled time with assigned partner
    • Follow the topic prompt and speak naturally in both languages
    • Ensure microphone is working and background noise is minimal
    • Complete the full recording duration (minimum 5 minutes per session)
    • Re-record sessions if flagged by the QC team

    Profile Card

    A personal profile card displays the speaker's avatar (initials), name, country, age range, gender, language badges, and time zone.

    Live Topics Preview

    Up to 3 available live topics are shown, each with title, category badge, language pair, and participant slots filled (e.g., '3/6'). Clicking navigates to the full sessions page.

    Upcoming Sessions Preview

    Upcoming sessions show topic title, scheduled date/time, language pair, and participant avatars (up to 4 shown with '+N' overflow).
